# 🐾 Dijital Hayvanım - React Native Oyunlaştırma Projesi

Bu uygulama, **AIgile Mobile Geliştiricileri** eğitimi kapsamında geliştirilmiş, seviye sistemi ve hayatta kalma mekaniklerine sahip bir sanal bebek (Tamagotchi) oyunudur.

## 🎯 Projenin Amacı ve Oyunlaştırma Özellikleri
Uygulama, temel bir mobil uygulamayı oyun dinamikleriyle birleştirerek kullanıcı etkileşimini artırmayı hedefler.
- **Seviye Sistemi (Leveling):** Yumurtadan başlayarak 6 farklı gelişim aşaması (Bebek, Canavar vb.).
- **Dinamik Ödüller (Achievements):** Seviye 4, 5 ve 6'da açılan Gözlük, Şapka ve Büyülü Küre aksesuarları.
- **Hayatta Kalma (Survival):** Zamanla azalan Açlık, Mutluluk ve Temizlik barları.
- **Oyun Sonu (Game Over):** İhtiyaçlar karşılanmadığında oyunun bitmesi ve yeniden başlama zorunluluğu.
- **Başarı Mesajı:** En üst seviyeye ulaşıldığında kullanıcıya sunulan tebrik ekranı.

## 🛠️ Kurulum ve Çalıştırma
Uygulamayı kendi bilgisayarınızda çalıştırmak için:
1. Projeyi bilgisayarınıza indirin: `git clone https://github.com/suedaknl/dijital-hayvanim.git`
2. Proje klasörüne girin: `cd dijital-hayvanim`
3. Gerekli paketleri kurun: `npm install`
4. Uygulamayı başlatın: `npx expo start`
5. Expo Go uygulaması ile QR kodu taratarak cihazınızda test edin.
